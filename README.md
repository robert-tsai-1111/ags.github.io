## AGT AI 基礎建設 規劃設計建置服務. 
  
AGT AI資料中心中團隊, 協助社群針AI專案協助規劃設計方案與部署運算儲存系統, 包括: 
   
* AI算力需求設計: 依應用計算CPU核心數, Memory容量, 依AI模型計算GPU快取容量頻寬,與使用案例所需要的節點/GPU數量. 
* 資料儲存需求設計: 依應用規模, 通訊協定(iSCSI,NFS), 資料形態特性(結構,物件), 平𡶶尖𡶶用戶規模來計算資料出入(IOPS), 資料頻寬 (Data Rate), 傳輸吞吐量 (Bandwidth)規格. 
* 機櫃與資料中心設計: 依節點重量體積計算機櫃空間與連線規格, 依節點電力冷卻需求計算資料中心基建規格, 實施現場檢查(Site Check),白空間(White Space)合規檢查. 
* 資料中心與企業網路整合設計: 依使用案例,資料內容(System, Application, PII), 傳輸規格要求(25Gbps, 100Gbps), 安全合規要求(ACL, ZeroTrust)規劃網路整合方案. 
* 部署與落地: 依架構規劃設計施工,測試,驗收.

## AGS BI 商業智慧規劃顧問服務. 
  
BI (Business Intelligence) 系統是一套將部門(社群))內外部資料, 轉化為可分析, 可視覺化, 可決策資訊的平台, 支援管理層與業務單位做出資料驅動決策 (Data-Driven Decision Making).

AGS BI 智慧團隊提供下列規劃顧問服務:

* 資料來源規劃: 營運系統如ERP,CRM, 電商系統如Shopify, 行銷系統如 Facebook,Tiktok, 資安系統如 Syslog, XDR等. 
* 資料模型規劃: 星型/雪花資料模型, 主鍵設計, 索引設計. 
* 資料擷取 (Extract, Ingest): 每日資料更新, 歷史資料保存. 
* 資料轉換與清洗 (Transform, Clenase). 
* 智慧資料載入生成 (Load): 主資料, 洞察資料, 可行動資料市集. 
* 可視化: 儀表板, 主動式資料通知, 互動式分析, 定期性報表. 
  
  
常見商業智慧方案例如產品推存, 客戶續約, 客戶流失分析, 異常偵測, 我們很樂意與您更深入地探討我們如何加速智慧來促成營運成功.
